### Option 2: Run Locally
If you prefer to run the notebook on your own machine:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/bochap-opensource/ngms-careerday.git](https://github.com/bochap-opensource/ngms-careerday.git)
    cd ngms-careerday
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Launch Jupyter:**
    ```bash
    jupyter notebook
    ```
